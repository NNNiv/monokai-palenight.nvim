## Install

## Packer
```lua
use "NNNiv/monokai-palenight.nvim"
```
## Add this to your init.lua
```lua
vim.cmd[[ colorscheme monokai-palenight ]]
``` 




## Supported Plugins
* Treesitter [recommended for better syntax highlighting]
* NvimTree
* LuaLine
* Treesitter
* Telescope
* GitSigns
* Todo Comments
